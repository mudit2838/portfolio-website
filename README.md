# Portfolio Website : https://portfolio-website-mk.vercel.app/

This is my personal portfolio website built with Next.js. It shows my skills, projects, and contact details.

## Features

- Clean and modern design  
- Responsive for all devices  
- About Me section  
- Skills section  
- Projects section  
- Contact section  
- Fast performance  

## Technologies Used

- Next.js  
- React.js  
- CSS / Tailwind CSS  

## How to Run

1. Clone the project  
2. Open the folder  
3. Install dependencies:

npm install

4. Start project:

npm run dev
